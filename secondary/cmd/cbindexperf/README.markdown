cbindexperf
===========

Bulk scan request perf reporting tool for secondary indexes

## Examples

    $ cbindexperf -cluster 127.0.0.1:9000 -configfile config.json -resultfile result.json

    $ cat config.json

       "Concurrency" : 1,
       "Clients": 1,
       "ScanSpecs" : [
          {
             "Type" : "All",
             "Id" : 1,

             "Limit" : 100,
             "Bucket" : "default"
          },
          {
             "Limit" : 0,
             "Low" : [
                "Aaron"
             ],
             "Repeat" : 0,
             "Type" : "Range",
             "Bucket" : "default",

                "Z"

             "Inclusion" : 3,
